Detailed Description

Eye specific data.

Field Documentation

bool Calibrated

Indicates whether the eye is calibrated. If this is false then the GazePoint is not valid for the eye.

bool Found

Indicates whether the eye is found in the image. If this is false then the rest of the data for the eye is not valid.

float PupilDiameter

The diameter of the pupil in millimeters. The resolution of this measurement is .02mm. If the eye is found and the pupil diameter is less than zero then the diameter could not be determined and it's value should not be used for that frame.
